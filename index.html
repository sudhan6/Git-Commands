Github Commands list

  1. pwd - file location
  2. ls - name of the files in the folder
  3. ls -a - files name in the folder (hidden files also)
  4. cd (file name) - change file location
  5. cd .. - back to the same file
  6. git add (file name) - add to the change panel
  7. git add . - adding the all changes to the staggered panel
  8. git commit -m "commit messg" - for git commit
  9. git remote show - to show the remote location
  10. git branch - to show the branch
  11. git version - to show the git version
  12. git log - to show history of all commit
  13. git push -u origin (branch name) - to push the command to git hub
  14. git remote add (repo url) - to add me remote
  15. git config --global user.name - to show user name
  16. git config --global user.email - to show email
  17. git config --global user.name "user name" - to add user name
  18. git config --global user.email "Email" - to show user email
  19. git branch (new branch name) - to create new branch
  20. git checkout (branch name) - to change the branch
  21. git checkout -b (branch name) - to create branch & even shit to the branch
  22. git branch -D (branch name) - to delete a branch
  23. git init - to add .git file in a folder
  24. rm -rf .git - to delete a .git file
